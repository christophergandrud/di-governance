Data

The constricted data set used in the paper's analyses can be found at:


https://github.com/christophergandrud/di-governance.github.com/blob/master/data_files/DI_analysis_data.csv



To load this data directly into R use the following commands (for more details see here):

            
            library(RCurl)
            library(foreign)

            url <- "https://raw.github.com/christophergandrud/di-governance.github.com/master/data_files/DI_analysis_data.csv"

            disproportionality.data <- getURL(url)                

            disproportionality.data <- read.csv(textConnection(disproportionality.data))
